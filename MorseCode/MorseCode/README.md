# Morse Code iOS App (SwiftUI)

This folder contains the SwiftUI source files and assets for the iOS app.

## How to open in Xcode
1. In Xcode, create a new iOS App project named `MorseCode` in the repo root.
2. Replace the generated Swift files with the ones in this folder.
3. Drag `Assets.xcassets` from this folder into the Xcode project (replace the default asset catalog).
4. Run on simulator or device.

## Files
- `MorseCodeApp.swift` – app entry point
- `ContentView.swift` – main UI
- `TrainerViewModel.swift` – game logic
- `MorseKeyView.swift` – input key UI
- `MorseAlphabet.swift` – morse map
- `TonePlayer.swift` – audio tone
- `HapticPlayer.swift` – haptics
- `Assets.xcassets` – colors
